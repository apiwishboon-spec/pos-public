About

POS System is a desktop point-of-sale application built with Python and Qt (PySide6). Designed for small businesses that need a simple, offline-capable POS with inventory tracking, receipt printing, and PromptPay QR payment support.

Built with Python, PySide6, and SQLite. Licensed under a proprietary license — see the license agreement for details.
